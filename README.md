# SkillBridge Portal

A Django-based Internship Portal where students can search, apply, and track internship applications.

## Features

- User Login & Authentication
- Internship Listings
- Apply for Internships
- My Applications Dashboard
- Django Admin Panel
- SQLite Database Integration

## Tech Stack

- Python
- Django
- HTML
- CSS
- SQLite
- Git & GitHub

## Project Screens

- Internship Listing Page
- Apply Internship Feature
- Application Tracking Page

## Installation

```bash
git clone https://github.com/Sejalpandit/skillbridge-portal.git
cd skillbridge
pip install -r requirements.txt
python manage.py runserver
```
